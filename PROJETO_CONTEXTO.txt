# Windows 7 Portfolio - Contexto do Projeto

## Visão Geral
Portfolio pessoal estilo Windows 7 construído com React. O projeto simula um desktop do Windows 7 funcional com janelas arrastáveis, menu iniciar, barra de tarefas e sistema de som.

## Stack Tecnológico
- React 18
- Phosphor Icons (@phosphor-icons/react)
- CSS Modules
- Vite (build tool)

## Funcionalidades Implementadas

### 1. Sistema de Janelas
- Janelas arrastáveis (draggable)
- Múltiplas janelas abertas simultaneamente
- Z-index para sobreposição
- Estados: minimized, maximized, normal

### 2. Sistema de Áudio
- Sons de click, hover, open, close
- Música de fundo (music.mp3)
- Controle de volume global
- wallpapers diferentes para quando mutado/unmutado

### 3. Sistema Bilingue (i18n)
- Suporte EN e PT
- Toggle de idioma no menu iniciar
- Função `t()` para traduções

### 4. Start Menu
- Glass morphism
- Botão iniciar com gradiente roxo
- Itens de menu estilizados

### 5. Sobre/About Me
- Foto de perfil (perfil.jpg)
- Posição ajustada: object-position: 50% 95%

## Referências de Código

### Como usar Phosphor Icons
```jsx
import { Icon } from '@phosphor-icons/react';
<Icon name="arrow-left" weight="bold" />
```

### Como usar traduções
```jsx
import { useLanguage } from './contexts/LanguageContext';
const { t } = useLanguage();
<button>{t('previous')}</button>
```
